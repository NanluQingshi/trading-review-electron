================================================================
             TradingReview - 安装与使用说明
================================================================

感谢您使用 TradingReview！
本软件由个人开发者构建，暂未购买企业签名证书，因此在首次安装时，
系统可能会提示“安全警告”或“文件已损坏”。

请放心，这是正常的系统机制，软件本身是安全无毒的。
请根据您的电脑系统，按照以下步骤操作即可正常使用。

----------------------------------------------------------------
🍎 macOS 用户 (Mac 电脑)
----------------------------------------------------------------

【情况 1：正常安装】
1. 双击打开下载的 .dmg 文件。
2. 将 TradingReview 图标拖入右侧的 Applications (应用程序) 文件夹。
3. 在“启动台”或“应用程序”中找到 TradingReview。
4. 【关键步骤】第一次打开时，请不要直接双击！
   -> 对着图标点击鼠标右键。
   -> 选择菜单中的“打开”。
   -> 在弹出的窗口中点击“打开”按钮。

【情况 2：提示“文件已损坏”或无法打开】
如果上述方法无效，或者系统提示“文件已损坏，建议移到废纸篓”，请使用我为您准备的一键修复工具：

1. 在下载的文件夹中找到 "fix-tradingreview.command" 文件。
2. 【关键步骤】对着该文件点击鼠标右键 -> 选择“打开”。
   (不要直接双击，可能会被拦截)
3. 在弹出的黑色终端窗口中，按照提示输入您的电脑开机密码。
   (注意：输入密码时屏幕上不会显示字符，这是正常的，输完直接回车即可)
4. 当显示“修复成功”后，关闭窗口，即可正常打开 TradingReview。

----------------------------------------------------------------
🪟 Windows 用户
----------------------------------------------------------------

1. 双击运行 "TradingReview Setup.exe" 安装包。

2. 如果出现蓝色的“Windows 已保护你的电脑”窗口 (SmartScreen)：
   -> 请点击文字链接 【更多信息】 (More info)。
   -> 点击右下角的按钮 【仍要运行】 (Run anyway)。

3. 等待安装完成后，软件会自动启动，桌面也会生成快捷方式。

----------------------------------------------------------------
❓ 常见问题
----------------------------------------------------------------

Q: 为什么会有这些安全提示？
A: 因为本软件是个人开发，没有向 Apple/Microsoft 缴纳昂贵的企业认证费用。
   这并不代表软件有病毒，请放心使用。

Q: 软件无法连接数据库或数据丢失？
A: 请确保您没有将软件安装在含有中文或特殊字符的路径下。
   建议保持默认安装路径。
