# 1.优化后的流程

```


plantuml
@startuml
title 公司外汇交易系统化流程活动图

start

:业务部门在线提交外汇交易申请;

partition "外汇交易管理系统" {

    :系统自动校验基础数据;
    :系统记录申请信息并生成流程编号;

    :触发线上审批流程;
}

partition "审批环节" {
    :财务经理在线审核额度;
    if (审核通过?) then (是)
        :高层在线审批确认;
    else (否)
        :系统自动驳回并通知申请人;
        stop
    endif
}

partition "交易执行阶段" {
    :交易员根据审批结果执行交易;
    :系统录入成交信息;
    :自动生成交易台账记录;
}

partition "交割与跟踪" {
    :系统自动提醒交割日期;
    :在线登记交割信息;
    :更新交易状态;
}

:系统生成完整交易记录;
:完成会计记账处理;

stop
@enduml
```
