================================================================
        TradingReview - Windows 安装与使用说明
================================================================

感谢您使用 TradingReview！
本软件由个人开发者构建，暂未购买企业签名证书，因此在首次安装时，
系统可能会提示"安全警告"。

请放心，这是正常的系统机制，软件本身是安全无毒的。
请按照以下步骤操作即可正常使用。

----------------------------------------------------------------
📥 安装步骤
----------------------------------------------------------------

【第一步：下载文件】
1. 下载 TradingReview Setup.exe 安装包

【第二步：运行安装程序】
1. 双击运行 "TradingReview Setup.exe" 安装包

【第三步：处理安全提示】
如果出现蓝色的"Windows 已保护你的电脑"窗口 (SmartScreen)：
-> 请点击文字链接 【更多信息】 (More info)
-> 点击右下角的按钮 【仍要运行】 (Run anyway)

【第四步：完成安装】
1. 等待安装程序完成
2. 安装完成后，软件会自动启动
3. 桌面上会生成 TradingReview 快捷方式

----------------------------------------------------------------
🚀 启动应用
----------------------------------------------------------------

【从桌面启动】
- 双击桌面上的 TradingReview 图标

【从开始菜单启动】
1. 点击开始菜单
2. 找到 TradingReview
3. 点击打开

【从安装目录启动】
1. 打开安装目录（默认：C:\Users\用户名\AppData\Local\Programs\trading-review）
2. 双击 TradingReview.exe

----------------------------------------------------------------
🔧 遇到问题？
----------------------------------------------------------------

【问题 1：Windows Defender 阻止安装】

1. 点击"更多信息"
2. 点击"仍要运行"
3. 如果仍然被阻止，请暂时关闭 Windows Defender 实时保护

【问题 2：杀毒软件报毒】

1. 将 TradingReview 添加到杀毒软件的白名单
2. 重新运行安装程序
3. 不同杀毒软件的操作方式略有不同，请参考杀毒软件的帮助文档

【问题 3：应用无法启动】

1. 确保您的 Windows 版本为 Windows 10 或更高
2. 检查是否安装了 .NET Framework（Windows 10 自带）
3. 尝试以管理员身份运行
4. 重新安装应用

【问题 4：应用闪退或无响应】

1. 重启计算机
2. 检查是否有其他程序冲突
3. 以管理员身份运行
4. 联系技术支持

----------------------------------------------------------------
💾 数据存储说明
----------------------------------------------------------------

TradingReview 的数据存储在以下位置：
%APPDATA%/TradingReview/

具体路径示例：
C:\Users\您的用户名\AppData\Roaming\TradingReview\

如果您需要备份或迁移数据，可以复制上述文件夹到其他位置。

查看数据存储位置的方法：
1. 打开 TradingReview 应用
2. 点击帮助菜单
3. 选择"打开数据文件夹"

----------------------------------------------------------------
❓ 常见问题
----------------------------------------------------------------

Q: 为什么会有这些安全提示？
A: 因为本软件是个人开发，没有向 Microsoft 缴纳昂贵的企业认证费用。
   这并不代表软件有病毒，请放心使用。

Q: 软件无法连接数据库或数据丢失？
A: 请确保您没有将软件安装在含有中文或特殊字符的路径下。
   建议保持默认安装路径。

Q: 如何卸载软件？
A: 1. 打开"控制面板"
   2. 选择"程序和功能"
   3. 找到 TradingReview
   4. 点击"卸载"
   如需删除数据，手动删除 %APPDATA%/TradingReview/ 文件夹。

Q: 应用更新后数据会丢失吗？
A: 不会。应用更新会保留所有您的交易数据。

Q: 可以安装到其他目录吗？
A: 可以。在安装过程中选择"自定义安装"，然后选择您想要的安装路径。
   但建议使用默认路径以避免兼容性问题。

Q: 支持哪些 Windows 版本？
A: 支持 Windows 10 及更高版本（包括 Windows 11）。
